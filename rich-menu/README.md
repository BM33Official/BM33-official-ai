# Rich Menu (แผงปุ่มลัดใต้ช่องแชท)

เมนู 6 ช่อง (ตาราง 3×2) — แตะแล้วส่งข้อความให้บอทตอบ (บอทจะตอบจาก FAQ/Gemini เหมือนพิมพ์เอง)

## ต้องเตรียม: รูป `rich-menu.png`
- ขนาด **2500 × 1686 px**, ไฟล์ **PNG**, ขนาด **≤ 1 MB**
- แบ่งเป็น 6 ช่องตามนี้ (ให้ตรงกับ `rich-menu.json`):

```
+----------------+----------------+----------------+
|  เงินรุ่น 💸    |  ตารางเรียน 📅  |  กิจกรรม 🎉    |   แถวบน (y 0–843)
+----------------+----------------+----------------+
|  ประกาศ 📢     |  คุยกับคน 🙋    |  โปรไฟล์รุ่น 🔗 |   แถวล่าง (y 843–1686)
+----------------+----------------+----------------+
   x 0–833          x 833–1667        x 1667–2500
```

แต่ละช่องกว้าง ~833px สูง 843px — ทำรูปให้ไอคอน/ข้อความอยู่กลางช่องพอดี

## ปุ่มแต่ละช่องทำอะไร (แก้ได้ใน `rich-menu.json`)
| ช่อง | แตะแล้ว |
|---|---|
| เงินรุ่น | ส่งข้อความ "เงินรุ่นเทอมนี้เท่าไหร่" |
| ตารางเรียน | ส่ง "ตารางเรียนสัปดาห์นี้" |
| กิจกรรม | ส่ง "กิจกรรมรุ่นเร็ว ๆ นี้มีอะไรบ้าง" |
| ประกาศ | ส่ง "ประกาศล่าสุดของรุ่น" |
| คุยกับคน | ส่ง "ขอคุยกับคนดูแลรุ่น" (บอท route ไปคนดูแล) |
| โปรไฟล์รุ่น | เปิดลิงก์ LINE (แก้ URL เป็นของจริงได้) |

> ปุ่มพวกนี้ส่ง "ข้อความ" ปกติ ให้ผลลัพธ์ดี ต้องมีคำถามพวกนี้ (หรือใกล้เคียง) อยู่ใน FAQ Sheet ด้วย

## ติดตั้ง
```bash
export LINE_CHANNEL_ACCESS_TOKEN="<access token ของ OA>"
bash rich-menu/setup-rich-menu.sh
```
สคริปต์จะ: สร้างเมนู → อัปรูป → ตั้งเป็น default ให้ทุกคน

## เปลี่ยน/ลบเมนูภายหลัง
- **เปลี่ยน:** แก้ `rich-menu.json` และ/หรือ `rich-menu.png` แล้วรันสคริปต์ใหม่ (ได้ id ใหม่ ตั้ง default ทับให้เอง)
- **ดูเมนูทั้งหมด:** `curl -H "Authorization: Bearer $LINE_CHANNEL_ACCESS_TOKEN" https://api.line.me/v2/bot/richmenu/list`
- **ลบ:** `curl -X DELETE -H "Authorization: Bearer $TOKEN" https://api.line.me/v2/bot/richmenu/<id>`
