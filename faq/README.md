# FAQ seed data

ไฟล์ CSV ตรงนี้คือ "ข้อมูลตั้งต้น" สำหรับ FAQ Sheet — Claude เตรียมให้จากเอกสารจริง
**บอทอ่านจาก Google Sheet สด (ผ่าน `SHEET_CSV_URL`) เท่านั้น** ไฟล์ในนี้ยังไม่ทำให้บอทตอบได้จนกว่าจะเอาเข้า Sheet

## ไฟล์
- `immunology-1007104.csv` — ตารางเรียนวิชาระบบภูมิคุ้มกันของมนุษย์ (Immunology 1007104) บล็อก 21 ก.ค.–19 ส.ค. 2569 → แปลงเป็น FAQ หมวด "วิชาการ"

## วิธีเอาเข้า Google Sheet (เลือกวิธีเดียว)
**วิธี A — Import (ง่ายสุด ได้คอลัมน์อัตโนมัติ):**
1. เปิด Google Sheet ที่ใช้เป็น FAQ → File → Import
2. แท็บ Upload → ลากไฟล์ `immunology-1007104.csv` เข้าไป
3. Import location = **Append rows to current sheet** (ต่อท้ายแถวเดิม) → Import data
4. ถ้าแถวหัว (หมวด/คำถาม/คำตอบ) ซ้ำกับที่มีอยู่แล้ว ลบแถวหัวที่ import มาเกินออก 1 แถว

**วิธี B — Copy/paste:**
1. เปิดไฟล์ CSV บน GitHub → กดดูแบบ Raw → เลือกทั้งหมด copy
2. ใน Sheet คลิกช่อง A ของแถวว่างถัดไป → Paste
3. ถ้าไม่แตกเป็นคอลัมน์ ให้ใช้ Data → Split text to columns → เลือกตัวคั่น "," (comma)
